# LINZA Agent Pack

This folder contains portable agent instructions for operating LINZA MCP.

Use `skills/linza-operator/SKILL.md` as the primary skill. The reference files
are optional context that an agent should load only when needed.

Contents:

```text
skills/linza-operator/SKILL.md
skills/linza-operator/references/workflows.md
skills/linza-operator/references/safety-policy.md
skills/linza-operator/references/tool-audience.md
```

The agent pack does not contain private paths, local vault data, SQLite files, or
generated reports.
